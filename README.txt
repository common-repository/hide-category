== Description ==

WC Hide Category is Open source Software. You can easily hide your Unnecessary/Private Category on the Shop page and Post Page.
Many Vendors And Ecommerce Shopkeepers want to hide some Posts and Shop Category In fact, WordPress and WooCommerce have the option to save products and posts as drafts! But this is not a good way and lot of time consuming. And when you try to save your Product/Post in the Draft, you can hide only on single Product/Post not Product/Post Category.
The project is following a two-phase process that will touch these two major pieces of WordPress.
1. Hide Product Category
2. Hide Post Catefgory
The following features are included in WC Hide Category:
* Easily install and setup only with One Click.
* Easily hide your Multiple Category in One Click.
* Multiple language support.
* Select multiple categories.
* Hide Category and Sidebar widgets on shop page
* Hide products categories on shop page
* Hide Post categories on post page


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wc-hide-category.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. From plugin after being acitivate click on settings tab and select category to hide and hite save.
2. After plugin activated, navigate to Woocommerce>Setting>Hide Category
